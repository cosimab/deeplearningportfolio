# deeplearningportfolio
This repository contains my portfolio submission for the Deep Learning module in the Data Science M.Sc. program at HAW Kiel. The objective of this project was to conduct a comprehensive deep learning experiment on a real-world image dataset.

Grade achieved: 100/100 points

* [`report.ipynb`]: The primary Jupyter Notebook containing all experiments.
* [`pitch.pdf`]: A pitch for a fictional customer as required by task 3.1
* [`portfolio_2025_wise.pdf`]: Official exam description detailing the tasks and evaluation criteria.
* [`assembled_waste_data.csv`]: The dataset I used, obtained from https://archive.ics.uci.edu/dataset/908/realwaste. The csv file contains the labels, including manual labelling I did for the plastic class. The images have to be downloaded from UCI in order to run the code.
